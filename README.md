## What I'm working on

I was planning to use this account for all ENI-related projects, but I ended up using it for AOE3-related projects instead. I'm currently working on:

- [Age of the World](https://forums.ageofempires.com/t/age-of-the-world-mod-for-definitive-edition-spotlight/210784), the biggest AOE3 Definitive Edition mod. I'm writing and maintaining AI scripts for every civilization in the mod.
- [Age of Pirates](https://github.com/thinotmandresy/age-of-pirates), another AOE3 Definitive Edition that adds new maps and new natives to the game, with lots of interesting features. My role is to write and maintain trigger scripts for certain gameplay features since many features are implemented via triggers in the mod.
- [Age of Empires III XS Language Support](https://github.com/thinotmandresy/age3xs), a VSCode extension that adds syntax highlighting and completion for the XS language used in Age of Empires III.

I'm looking for more "serious", real-world projects to work on, and I'm very open to collaborate. If you have any ideas, feel free to reach out!
